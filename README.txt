German Roleplay ist ein von Xendom Rayden zwischen Ende 2014 und Anfang 2016 überarbeitetes und erweitertes Vio-Extended. Als Hauptstadt wurde Los Santos gewählt.

Die Arbeiten an diesem Gamemode wurden zum Großteil eingestellt. Es werden lediglich noch Fehlermeldungen, Anmerkungen und Pull requests bearbeitet. Sollte man ein ausgearbeitetes Reallife / Roleplay Script benötigen, empfehle ich meinen Gamemode "Berkshire County Reloaded".
